Give a followed user a short nickname.  After setting an alias, you can
use it anywhere you would use @name or a user ID.

HOW TO READ THIS HELP
  The examples below show real commands you can copy and paste.
  Everything you should type is shown — just replace the example values
  with your own.

    <angle brackets>   Replace with your own value.  Do NOT type the brackets.
                       peerpedia article show <id>
                       → you actually type:  peerpedia article show abc12345

    [square brackets]  This part is optional.  You can skip it entirely.
                       peerpedia article list [--mine]
                       → you can type:  peerpedia article list
                       → or:            peerpedia article list --mine

    --double-dashes    A "flag" (option) that changes what the command does.
                       --title means "the title comes next"
                       --scores means "the scores come next"
                       Some flags take a value (--title "My Paper")
                       Others work alone (--mine, --force, --feed).

  The prompt character > or $ in examples means "type this in your terminal."
  Don't type the prompt itself.


EXAMPLES
  # Set an alias for a user by @name
  peerpedia alias set @pierre_curie pierre
  # → Now "pierre" is a shortcut you can type instead of @pierre_curie.

  # Set an alias by user ID
  peerpedia alias set abc12345 marie
  # → Works the same way.  Any unique identifier for the user is fine.

  # Now use the alias in other commands
  peerpedia follow marie
  peerpedia review invite my-paper --user pierre
  # → Aliases work everywhere.  Instead of typing @pierre_curie every
  #    time, just type "pierre".

  # Set short aliases for your frequent collaborators
  peerpedia alias set @albert_einstein al
  peerpedia alias set @niels_bohr niels
  peerpedia alias set @werner_heisenberg werner
  # → Pick whatever nickname you will remember.  Only you see them.

WHAT ARE ALIASES
  • Aliases are local to your machine — only you see them.  Other
    users still see each other's real @names.
  • They save typing and make commands more readable.
  • You can use an alias anywhere PeerPedia asks for a user: @name,
    user ID, or alias all work.
  • To see all your aliases:  peerpedia alias list

FLAGS
  --json    Output the result as JSON instead of plain text.
            Useful for scripts and automation.

  --rich    Force human-readable colored output (default when
            you type the command directly).

SEE ALSO
  alias list         See all your aliases
  alias remove       Remove an alias
  follow             Follow a user
