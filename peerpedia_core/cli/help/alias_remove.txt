Remove an alias you previously set for a user.  The alias is deleted from
your local list — the user is not affected in any way.

HOW TO READ THIS HELP
  The examples below show real commands you can copy and paste.
  Everything you should type is shown — just replace the example values
  with your own.

    <angle brackets>   Replace with your own value.  Do NOT type the brackets.
                       peerpedia article show <id>
                       → you actually type:  peerpedia article show abc12345

    [square brackets]  This part is optional.  You can skip it entirely.
                       peerpedia article list [--mine]
                       → you can type:  peerpedia article list
                       → or:            peerpedia article list --mine

    --double-dashes    A "flag" (option) that changes what the command does.
                       --title means "the title comes next"
                       --scores means "the scores come next"
                       Some flags take a value (--title "My Paper")
                       Others work alone (--mine, --force, --feed).

  The prompt character > or $ in examples means "type this in your terminal."
  Don't type the prompt itself.


EXAMPLES
  # Remove an alias by @name
  peerpedia alias remove @pierre_curie
  # → If you had set "pierre" as an alias for @pierre_curie, it is gone.

  # Remove by user ID
  peerpedia alias remove abc12345
  # → Same effect.  Any identifier that matches the user works.

  # Remove by alias name
  peerpedia alias remove pierre
  # → You can also use the alias itself to refer to the user you want
  #    to un-alias.  The alias is removed after this.

  # Clean up an alias you no longer need
  peerpedia alias remove @previous_colleague
  # → Maybe that collaborator left your field, or you switched to using
  #    their full @name.  The alias is gone, nothing else changes.

WHAT HAPPENS WHEN YOU REMOVE AN ALIAS
  • The alias is deleted from your local list.
  • The user is unaffected — they keep their @name and everything else.
  • You just can't use that nickname to refer to them anymore.
  • The original @name still works for all commands.

FLAGS
  --json    Output the result as JSON instead of plain text.
            Useful for scripts and automation.

  --rich    Force human-readable colored output (default when
            you type the command directly).

SEE ALSO

  alias list         See all your aliases
