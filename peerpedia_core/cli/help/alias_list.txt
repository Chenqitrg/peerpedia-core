List all the local aliases you've set for users.  Shows each alias and
which user it points to.

HOW TO READ THIS HELP
  The examples below show real commands you can copy and paste.
  Everything you should type is shown — just replace the example values
  with your own.

    <angle brackets>   Replace with your own value.  Do NOT type the brackets.
                       peerpedia article show <id>
                       → you actually type:  peerpedia article show abc12345

    [square brackets]  This part is optional.  You can skip it entirely.
                       peerpedia article list [--mine]
                       → you can type:  peerpedia article list
                       → or:            peerpedia article list --mine

    --double-dashes    A "flag" (option) that changes what the command does.
                       --title means "the title comes next"
                       --scores means "the scores come next"
                       Some flags take a value (--title "My Paper")
                       Others work alone (--mine, --force, --feed).

  The prompt character > or $ in examples means "type this in your terminal."
  Don't type the prompt itself.


EXAMPLES
  # Show all aliases
  peerpedia alias list
  # → pierre    → abc12345
  # → marie     → def67890
  # → al        → a1b2c3d4

  # After setting several aliases, see them all at once
  peerpedia alias set @albert_einstein al
  peerpedia alias set @niels_bohr niels
  peerpedia alias set @marie_curie marie
  peerpedia alias list
  # → al     → a1b2c3d4
  # → niels  → e5f6g7h8
  # → marie  → i9j0k1l2

  # Check whether you already have an alias for someone
  peerpedia alias list
  # → Scan the list.  If you don't see the person you need, add one
  #    with:  peerpedia alias set @name nickname

  # Remember what your own aliases are after not using them for a while
  peerpedia alias list
  # → Output shows alias name on the left, user ID on the right.
  #    Use the alias in any command that asks for a user.

WHAT THE OUTPUT SHOWS
  • Left column: the alias name you chose (e.g. "pierre", "marie").
  • Right column: the user's ID (a short identifier like abc12345).
  • If you don't recognize an ID, look them up with:
    peerpedia account search <id>

WHEN ALIASES ARE HELPFUL
  • You frequently type the same @name in commands.
  • A user's @name is long or hard to remember.
  • You work with the same small group of collaborators regularly.
  • You want short, memorable nicknames for your team.

FLAGS
  --json    Output the list as JSON instead of the table format.
            Useful for scripts and automation.

  --rich    Force human-readable colored output (default when
            you type the command directly).

SEE ALSO
  alias set          Create an alias
  alias remove       Remove an alias
