Rate a reviewer's helpfulness.  This contributes to the reviewer's
reputation score on the network.

HOW TO READ THIS HELP
  The examples below show real commands you can copy and paste.
  Everything you should type is shown — just replace the example values
  with your own.

    <angle brackets>   Replace with your own value.  Do NOT type the brackets.
                       peerpedia article show <id>
                       → you actually type:  peerpedia article show abc12345

    [square brackets]  This part is optional.  You can skip it entirely.
                       peerpedia article list [--mine]
                       → you can type:  peerpedia article list
                       → or:            peerpedia article list --mine

    --double-dashes    A "flag" (option) that changes what the command does.
                       --title means "the title comes next"
                       --scores means "the scores come next"
                       Some flags take a value (--title "My Paper")
                       Others work alone (--mine, --force, --feed).

  The prompt character > or $ in examples means "type this in your terminal."
  Don't type the prompt itself.


EXAMPLES
  # Rate a reviewer as very helpful (5 out of 5)
  peerpedia review rate abc12345 --reviewer @pierre --helpfulness 5
  # → Score recorded. @pierre's reputation increases.

  # Rate by user ID
  peerpedia review rate abc12345 --reviewer def67890 --helpfulness 4
  # → Same as above, but using the reviewer's user ID instead of @name.

  # What helpfulness scores mean
  #   1 — Not helpful (off-topic, vague, or unconstructive)
  #   2 — Somewhat helpful (minor useful points)
  #   3 — Helpful (solid feedback with specific suggestions)
  #   4 — Very helpful (detailed, constructive, actionable)
  #   5 — Exceptionally helpful (thorough, insightful, transformative)
  # A score of 3 is a good, solid review. Reserve 5 for truly
  # exceptional feedback that improved your paper significantly.

  # Can you change your rating?
  peerpedia review rate abc12345 --reviewer @pierre --helpfulness 4
  # → If you already rated this reviewer, the new score overwrites
  #   the old one. You can change your rating at any time — there is
  #   no limit on how many times you can update it.

  # What happens if you rate someone who hasn't reviewed your paper
  peerpedia review rate abc12345 --reviewer @einstein --helpfulness 3
  # → If @einstein hasn't reviewed this paper, PeerPedia says:
  #   "User @einstein has not reviewed this paper. Only reviewers
  #   of this paper can be rated."

RULES
  • Only the paper's author (or maintainers) can rate reviewers.
  • Helpfulness is scored from 1 (not helpful) to 5 (exceptionally
    helpful).
  • You can change your rating at any time — the new score replaces
    the old one.
  • Ratings are anonymous — the reviewer sees their total score but
    not who gave what.

FLAGS
  Each flag is shown in the examples above.  This reference explains them
  in full — come back after you have tried the examples.

  --reviewer <user>    Who you are rating.  Can be an @name (like
                       @pierre), a full user ID, or a partial ID.
                       Default: (required, no default).  Shown in all
                       examples above.

  --helpfulness <n>    Your rating from 1 (not helpful) to 5
                       (exceptionally helpful).  See the "What
                       helpfulness scores mean" example for the
                       full scale.  Default: (required, no default).
                       Shown in all examples above.
