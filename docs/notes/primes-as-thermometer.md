## 0. 一个孩子的问题

你有一个烧水壶。你把水从 0°C 加热到 100°C。在 99.9°C，水平静如镜。在 100°C，水剧烈沸腾，气泡从每一个尺度同时涌出——微观的气泡核、中观的湍流涡、宏观的蒸汽柱。在临界点，所有尺度同时被激活。

这是相变。

你有一个素数序列：2, 3, 5, 7, 11, 13, 17, 19, ... 一直到无穷。

在 p=2，你只有两个元素。代数结构粗糙，可能性稀少——像一个被冻结的系统。在 p=1000003，你有百万个元素，结构精细得接近连续。在 p→∞，你得到特征零——光滑的复代数几何，有无穷无尽的精细结构。

这也是相变吗？

这个问题听起来像胡言乱语。素数不是温度。Spec(Z) 不是水壶。模形式不是比热。

但也许——仅仅是也许——**它们是同一类东西，穿在不同的衣服里。**

这篇笔记试图认真对待这个想法。不把它当作一个隐喻，而当作一个**物理理论的原型**——有精确的对应关系、可计算的预测、以及指明在哪里挖的铲子标记。

这是自然哲学，不是数学。它可能全是错的。但好的错误比正确的无聊更有价值。

## 1. 实验：数论中的「热力学」行为

在提出理论之前，让我们看一些原始数据。

### 1.1 素数分布与随机矩阵

素数在整数中的分布看起来「随机」，但不是独立的随机——它们相互排斥（两个素数不能相邻，除了 (2,3)）。这种排斥有多强？

Montgomery (1973) 计算了 Riemann zeta 函数零点之间的 pair correlation。他得到的公式是：

\[
1 - \left( \frac{\sin \pi u}{\pi u} \right)^2
\]

这恰好是 Gaussian Unitary Ensemble (GUE) 随机矩阵本征值的 pair correlation。

这不是偶然。Odlyzko 在 1980 年代用超级计算机验证了十亿个零点——完美匹配 GUE 统计。

换句话说：**Riemann zeta 零点的统计行为与一个大的随机 Hermitian 矩阵的本征值完全相同。**

物理学家看到这个会立刻认出：**这是量子混沌系统的能谱统计。** Bohigas–Giannoni–Schmit 猜想（1984）：量子混沌系统的能谱总是展现出随机矩阵统计。如果 zeta 零点表现出 GUE 统计，那么 zeta 零点应该可以解释为某个量子混沌 Hamiltonian 的本征值。

这就是 Hilbert–Pólya 猜想的物理版本：**存在一个 Hermitian 算符，其本征值是 zeta 零点的虚部。**

但等等——这说的不只是「某个算符」。一个具有 GUE 能谱统计的系统在物理上是非常特殊的：它是量子混沌的、没有时间反演对称性的、所有经典周期轨道都 unstable 的。什么样的「算术运动」会产生这样的谱？

### 1.2 模形式系数与态密度

看模形式：
\[
\Delta(q) = q \prod_{n=1}^\infty (1 - q^n)^{24} = \sum_{n=1}^\infty \tau(n) q^n
\]
\[
= q - 24q^2 + 252q^3 - 1472q^4 + 4830q^5 - 6048q^6 + \cdots
\]

Ramanujan 的 \(\tau(n)\) 是整数。Ramanujan 注意到它们满足：
- \(\tau(mn) = \tau(m) \tau(n)\) 当 \(\gcd(m,n) = 1\)（可乘性）
- \(\tau(p^{k+1}) = \tau(p)\tau(p^k) - p^{11}\tau(p^{k-1})\)（递归）

更重要的是：对于素数 \(p\)，\(|\tau(p)| \le 2p^{11/2}\)（Ramanujan 猜想，Deligne 1974 证明）。

物理学家看到乘性 + 递归 + 增长率界会想到什么？

**如果是物理系统，\(\tau(n)\) 是在计数。** 计数什么？拥有某种对称性的系统中，能级 \(n\) 上的态数量。

乘性意味着：态空间在素数因式分解下张量化。\(\tau(mn) = \tau(m)\tau(n)\) 意味着「\(mn\) 的态空间」 = 「\(m\) 的态空间」 ⊗ 「\(n\) 的态空间」——这是**独立子系统**的张量积结构，就像两个独立粒子的 Hilbert 空间是各自 Hilbert 空间的张量积。

递归关系 \(\tau(p^{k+1}) = \tau(p)\tau(p^k) - p^{11}\tau(p^{k-1})\) 是什么？这是 Fourier 系数必须满足的**一个两体相互作用**——态不是完全自由的，素数幂之间有「相互作用」，强度受 \(p^{11}\) 控制。

\(\tau(n)\) 在计数的东西有一个内部动力学——不是完全自由的，有约束。

### 1.3 VOA 分次维数的整性

更一般的现象：**任何有理共形场论（CFT）的 VOA 分次维数都是正整数。** 以 Virasoro 极小模型 (3,4)（Ising 模型）为例：

\[
\chi_{(1,1)}(q) = \frac{1}{2} q^{-1/48} \left( \prod_{n=0}^\infty \frac{1}{1 + q^{n+1/2}} + \prod_{n=0}^\infty \frac{1}{1 - q^{n+1/2}} \right)
\]
\[
= q^{-1/48}(1 + q^2 + q^3 + 2q^4 + 2q^5 + 3q^6 + \cdots)
\]

每一个系数都是非负整数。为什么？目前的标准回答是：「因为特征标是模形式，模形式的 Fourier 系数在这个特殊情况下恰好是整数。」但这不是解释——这是在重命名问题。

**为什么一个代数对象（VOA）的分次维数必须是非负整数？** 从定义看，分次维数是向量空间的维数——当然是整数。但为什么 VOA 的分次结构必须与模形式结构「恰好匹配」？

如果把 VOA 看作一个物理系统（CFT 的手征部分），这个问题就消失了：**分次维数是态密度。态密度当然是正整数——你在计数物理态。** 模形式的整性不是巧合——它是「态计数 ⇒ 整性」这个物理原理在数学对象上的表观。

那么问题变成了：**计数的是什么态？**

## 2. 思想：素数 = 温度

### 2.1 重整化群流的基本图像

在统计力学中，研究一个物理系统意味着研究它如何随温度变化：

- **低温**（\(T \approx 0\)）：系统冻结在基态附近。只有最长的波长被激活——关联长度短，能区分的结构少。
- **升温**：越来越多的短波长模被热涨落激活。系统的有效自由度增加。关联长度增长。
- **临界温度** \(T_c\)：关联长度发散。所有波长——从微观到宏观——同时被激活。系统获得尺度不变性（scale invariance）。在 \(T_c\)，系统成为一个共形场论（CFT）。

从 \(T < T_c\) 到 \(T = T_c\) 的过程由重整化群（RG）描述。RG 的核心洞见：**改变温度相当于改变你观察系统的有效分辨率。** 升温 = 提高分辨率，看到更多的微观自由度。降温 = 降低分辨率，把微观自由度「积分掉」（coarse-grain）。

### 2.2 算术的 RG 流

现在看素数。

从 \(\mathbb{F}_p\) 的角度看整数方程。\(p\) 越小，域中的元素越少。在 \(\mathbb{F}_2\) 上，只有 \(\{0,1\}\)。很多在特征 0 中不同的对象在特征 2 中坍缩成同一个东西。只有最粗糙的结构能存活。

\(p\) 增大，域扩大。更多的点意味着更精细的区分能力。在 \(\mathbb{F}_p\) 上不同的对象在特征 0 可能仍然不同。

当 \(p \to \infty\)，域变成特征 0——\(\mathbb{Q}, \mathbb{R}, \mathbb{C}\)。有无穷多个元素。结构可以任意精细。

**这听起来很像温度：**

| 热力学 | 算术 |
|---|---|
| 温度 \(T\) | 素数 \(p\)（或 \(\log p\) 作为能量尺度） |
| 低温 \(T \approx 0\) | 小素数 \(\mathbb{F}_2, \mathbb{F}_3\) |
| 高温（临界）\(T = T_c\) | 大素数极限 \(p \to \infty\)（特征 0） |
| 升温 = 激活更多热涨落模 | \(p\) 增大 = 激活更多算术结构 |
| 基态 = 所有涨落冻结 | \(\mathbb{F}_1\) = 只有「存在」没有「差异」 |
| 临界点 = 所有尺度涨落共存 | \(\mathbb{C}\) = 所有算术结构被激活 |

在热力学中，当你从 \(T=0\) 升温到 \(T=T_c\)，系统的有效理论在每一个尺度上变化——RG 流。在算术中，当你沿着素数谱从 \(\mathbb{F}_2\) 走到 \(\mathbb{C}\)，你看到的是**算术结构的 RG 流**。

关键的主张不是类比——是**机制同一性**：

> 素数 \(p\) **就是**一个离散的温度参数。Spec(Z) **就是**系统的相图。从有限特征到特征零的过渡 **就是**重整化群流向临界点的过程。

### 2.3 为什么 log p 是能量尺度？

\(\log p\) 自然地出现为尺度参数：

- 素数定理：\(\pi(x) \sim x/\log x\)——素数分布的「密度」由 \(\log x\) 决定。\(\log x\) 是一个自然尺度。
- Hasse–Weil 界：\(\mathbb{F}_p\) 上曲线的有理点个数约为 \(p + 1\)，偏差受 \(2g\sqrt{p}\) 控制。平方根 \(\sqrt{p}\) 是 Frobenius 本征值的模。本征值的对数 \(\frac{1}{2}\log p\) 是自然的作用量/能量参数。
- AdS/CFT 和 p-进 AdS/CFT 中，径向坐标（重整化群尺度）通常取对数形式。

所以我们设定：**温度 \(k_B T_p \propto \log p\)。** 小素数 = 低能量尺度 = 只有长波模被激活。大素数 = 高能量尺度 = 短波模也活跃。

## 3. 词典：模形式 = 比热，特征标 = 配分函数

### 3.1 什么是比热？

当一个物理系统被加热时，它的内能 \(U(T)\) 随温度变化。比热 \(C(T) = dU/dT\) 测量的是：「在这个温度点，有多少自由度刚刚被解冻？」

更精确地说，\(C(T)\) 告诉你在温度 \(T\) 附近，有多少微观态开始对热力学有贡献。在临界点附近，\(C(T) \sim |T - T_c|^{-\alpha}\) 发散——在临界点上，**无穷多个自由度同时被激活**。

### 3.2 什么是模形式？

一个权 \(k\) 的模形式 \(f: \mathbb{H} \to \mathbb{C}\) 满足：
\[
f\left(\frac{a\tau + b}{c\tau + d}\right) = (c\tau + d)^k f(\tau), \quad \begin{pmatrix} a & b \\ c & d \end{pmatrix} \in \text{SL}(2,\mathbb{Z})
\]

它有 Fourier 展开：
\[
f(\tau) = \sum_{n=0}^\infty a_n q^n, \quad q = e^{2\pi i \tau}
\]

系数 \(a_n\) 是整数（对 Hecke 特征形式而言）。

模形式出现在 CFT 中作为**特征标**：\(\chi(\tau) = \operatorname{tr} q^{L_0 - c/24}\)。这是 CFT 的**配分函数在 torus 上的值**——\(L_0\) 是 scaling 算符，\(q^{L_0}\) 对态空间求迹。

所以：**模形式 = CFT 的 torus 配分函数。** 这已经是标准物理。

### 3.3 如果素数 = 温度，模形式是什么？

Torus 配分函数在二维 CFT 的意义：取一个周长为 \(2\pi\) 的圆，沿长度 \(\text{Im}(\tau)\) 卷成一个环面。\(\tau\) 的虚部控制环面的长宽比（即「温度方向」的长度）。\(q = e^{2\pi i \tau}\) 中，\(\text{Im}(\tau)\) 是环面的「时间」长度——温度 \(T \propto 1/\text{Im}(\tau)\)。

所以 \(q\) 展开是低温展开：\(q \to 0\) 时 \(\text{Im}(\tau) \to \infty\)，环面变得极长（极低温度）。系数 \(a_n\) 计数在「能级」 \(n\) 上有多少个态。

但现在，加入算术维度：

在「算术温度」轴上，每个素数 \(p\) 对应一个特定的能量尺度。在温度 \(p\)（即 \(\log p\) 的能量），我们只关注算术系统在这一尺度上的「态」。这些态的配分函数是：

\[
Z_p(q) = \sum_n a_n(p) q^n
\]

对于每个 \(p\)，\(a_n(p)\) 是这个系统在素数 \(p\) 的「算术温度」下，能级 \(n\) 的态密度。

现在关键来了：**当一个系统在它的临界温度（\(p = \infty\)，特征 0）时，所有尺度的态同时被激活，配分函数成为通常的模形式：**

\[
Z_\infty(q) = \lim_{p \to \infty} Z_p(q) = f(\tau)
\]

而 **\(Z_p(q)\) 本身是一个「截断的」模形式**——它只包含在能量尺度 \(\log p\) 以下被激活的态。

这听起来很玄。但 Hecke 特征形式的代数结构恰好提供了一个精确版本：

如果 \(f\) 是一个 Hecke 特征形式（归一化 \(a_1 = 1\)），则：

\[
a_{mn} = a_m a_n \quad \text{当 } \gcd(m,n)=1
\]
\[
a_{p^{k+1}} = a_p a_{p^k} - \chi(p) p^{k-1} a_{p^{k-1}}
\]

在物理图像中：
- **可乘性** \(a_{mn} = a_m a_n\)（对互素的 m,n）= 态空间按素数因子「张量化」。当 m 和 n 互素时，它们的素因子集合不相交——意味着「不同素数的自由度」是独立子系统，整体的态数 = 各自态数的乘积。这恰好是量子力学中独立子系统的张量积结构：\(\mathcal{H}_{mn} \cong \mathcal{H}_m \otimes \mathcal{H}_n\)。
- **递归关系** = 同一素数内部的「动力学」。\(p^{k-1}\) 项是素数幂能级之间的两体相互作用——它不是自由的，有一个随 p 增大而增长的耦合常数。

**需要澄清**：这不是说单个系数 \(a_n\) 被分解为素数因子的贡献（如 \(a_6 = a_2 \cdot a_3\) 是系数之间的等式，而非 \(a_6\) 的因子分解）。物理图像需要更精细的表述：**态空间本身由 Hecke 算子的公共特征基张成，每个素数 p 对应一个 Hecke 算子 T_p，态空间按 T_p 的特征值分解。** 算术 RG 流不是"a_n 在变化"，而是"每个素数的 Hecke 算子在不同温度下激活了态空间的不同子空间"。

更精确的表述留待 §7（在何处下铲子）中的项目 B 和 C 来发展。

### 3.4 比热 = 态密度；模权 = 反常电荷

模形式的权 \(k\) 控制模变换下的协变性：
\[
f(-1/\tau) = \tau^k f(\tau)
\]

从物理角度，权 \(k\) 与什么有关？需要先做一个重要的区分：

- **中心荷 c**：控制应力-能量张量的两点函数，决定态密度的渐近行为（Cardy 公式），以及 Virasoro 代数的中心扩张。
- **模权 k**：控制在模变换下配分函数的协变性（一个手征分量的权）。

在 CFT 中，**完整的 torus 配分函数是模不变的（权 0）**——左手征与右手征分量的权互相抵消：\(Z(\tau, \bar{\tau}) = \sum_{h,\bar{h}} N_{h,\bar{h}} q^{h-c/24} \bar{q}^{\bar{h}-c/24}\) 在 \(\tau \to -1/\tau\) 下不变。单个手征特征标（如 Virasoro 特征标）的权来自共形维数的分数部分，与中心荷 c 无直接关系。

所以模形式的权 k 不应直接被类比为"沿 RG 流的反常积分"——中心荷 c 才是那个角色。更准确的三层对应是：

| 物理量 | 数学量 | 关系 |
|---|---|---|
| 态密度 \(g(n)\) | Fourier 系数 \(a_n\) | \(a_n\) 计数能级 n 的态数 |
| 中心荷 \(c\) | （待定：可能与 zeta 的某种整体不变量相关） | 控制态密度的渐近增长 |
| 模权 \(k\) | 模变换的协变权重 | 控制特征标的模变换行为 |

在「素数 = 温度」的图像中：**中心荷 c 应该是沿素数 RG 流累积的反常总量**——类似于 Wess–Zumino 项的积分。模权 k 则是这个反常在手征分量上的投影，与具体的态空间截断方式有关。

一个推测：中心荷 c 可能与 \(\zeta(s)\) 在 \(s=1\) 的留数（或更一般地，与 L-函数在中心点的值）相关——这些是"全局"不变量，不依赖于任何单个素数的选择。确定这个对应是 §7 项目的关键目标之一。

## 4. Riemann zeta 零点 = CFT 的 scaling dimensions

### 4.1 谱-迹对偶

在量子力学中：

- **谱**：Hamiltonian \(H\) 的本征值 \(E_n\)。这些告诉你系统可以处于什么能量。
- **迹/配分函数**：\(Z(\beta) = \operatorname{tr} e^{-\beta H} = \sum_n e^{-\beta E_n}\)。这是对谱的 Laplace 变换。

谱 ←→ 配分函数是一对 Fourier/Laplace 对偶。知道谱 ⇒ 可以算配分函数。知道配分函数的所有奇异行为 ⇒ 可以还原谱。

在 CFT 中，谱是 scaling dimensions \(\Delta_n = h_n + \bar{h}_n\)。在 torus 上的配分函数（特征标）是：
\[
Z(\tau) = \sum_n q^{\Delta_n - c/24} \bar{q}^{\bar{\Delta}_n - \bar{c}/24}
\]
其中 \(q = e^{2\pi i \tau}\)。这个配分函数是模形式。

在数论中，Riemann zeta 函数的非平凡零点 \(\rho = 1/2 + i\gamma\) 形成了一个谱。

自然的问题是：**什么系统的配分函数以 zeta 零点为其谱？**

### 4.2 Weil 显式公式 = 迹公式

Weil 的显式公式（explicit formula）是数论中的「迹公式」。对于 test function \(h\)：

\[
\sum_\gamma \hat{h}(\gamma) = 2h\left(\frac{i}{2}\right) - \sum_p \sum_{m=1}^\infty \frac{\log p}{p^{m/2}} \left[ h(m \log p) + h(-m \log p) \right] - \int_{-\infty}^\infty \frac{\Gamma'(1/4 + \pi i t/2)}{\Gamma(1/4 + \pi i t/2)} h(t) dt
\]

其中 \(\gamma\) 跑遍所有非平凡零点的虚部（即 \(\rho = 1/2 + i\gamma\)）。

看这个公式的结构：

**左边** = 对所有非平凡零点的求和 = **谱侧**

**右边** = （对平凡零点的贡献）+（对所有素数幂的求和）+（Gamma 因子的贡献）= **几何侧 + 边界项**

素数 \(p\) 出现在几何侧的求和 \(\sum_p \sum_m \frac{\log p}{p^{m/2}} h(m \log p)\) 中。这里的 \(\log p\) 是自然尺度——恰好是我们在第 2 节中推断的「能量尺度」。

Weil 公式的结构是：
\[
\sum_{\text{零点}} \text{（test function 的 Fourier 变换）} = \sum_{\text{素数}} \text{（test function 的某种积分）}
\]

这正是**迹公式**的普遍形式！在量子混沌中，Gutzwiller 迹公式将 Hamiltonian 的谱（对所有周期轨道求和）表达为系统所有经典周期轨道的作用量求和。素数在 Weil 公式中扮演的角色，与**经典周期轨道**在 Gutzwiller 迹公式中的角色完全相同。

**所以素数 = 经典周期轨道。** 在 \(p \to \infty\)（连续极限）时，所有周期轨道同时贡献，迹变成 CFT 的特征标。

### 4.3 函数方程 = 模不变性

Riemann zeta 函数满足著名的函数方程。定义完备的 zeta 函数：

\[
\xi(s) = \frac{1}{2} s(s-1) \pi^{-s/2} \Gamma\left(\frac{s}{2}\right) \zeta(s)
\]

则：

\[
\xi(s) = \xi(1 - s)
\]

这个等式在结构的深度上**就是模不变性**。为什么？

在 CFT 中，torus 配分函数 \(Z(\tau)\) 在模变换 \(\tau \to -1/\tau\) 下不变（或协变）。τ 的虚部是 torus 的「空间」与「时间」周长之比——等价于温度的倒数。模变换 \(\tau \to -1/\tau\) 交换了空间与时间方向。

现在看 zeta 函数的函数方程。通过 Mellin 变换：

\[
\xi(s) = \int_0^\infty \left( \theta(it) - 1 \right) t^{s/2} \frac{dt}{t}
\]

其中 \(\theta(z) = \sum_{n=-\infty}^\infty e^{\pi i n^2 z}\) 是 Jacobi theta 函数。

而 Jacobi theta 函数满足**精确的模变换律**：

\[
\theta(-1/\tau) = \sqrt{-i\tau} \, \theta(\tau)
\]

所以 zeta 函数的函数方程 s ↔ 1-s **等价于** theta 函数的模变换 \(\tau \to -1/\tau\)。函数方程不是 zeta 的特殊性质——它是 theta 函数模不变性的影子，通过 Mellin 变换投射到 L-函数空间。

这意味着：
- **s ↔ 1-s 对称性 = τ ↔ -1/τ 对称性**
- **临界线 Re(s) = 1/2 = 模变换下的不动点**（因为 |τ| = 1, Re(τ) = 0 映射到自身）
- **零点在临界线上 = 谱在模变换下对称**

如果「素数 = 温度」，那么**函数方程就是系统在高温-低温对偶下的不变性**——这正是统计力学中 Kramers–Wannier 对偶的数学结构。在临界点（Re(s) = 1/2），系统同时在其自身的高温-低温对偶下不变——这是临界性的定义性质。

这条线索将 Riemann Hypothesis 重新表述为：**算术系统的 Kramers–Wannier 对偶不动点恰好落在「物理」谱上。** 这在概念上令人满意，但技术上的缺口仍然是：需要找到一个直接的配分函数定义——其 Mellin 变换给出 ξ(s)——而不只是通过 theta 函数的中介。这个缺口对应于找到一个「天然的」量子系统，其配分函数直接与 ξ(s) 对偶。

### 4.4 猜想的精确陈述

> **核心猜想**：存在一个中心荷 \(c\) 为无穷大（或极大）的共形场论——称为「算术 CFT」——定义在 \(\text{Spec}(\mathbb{Z})\) 的某个几何实现上，其 chiral algebra 在 torus 上的配分函数由 Riemann zeta 函数（或完备的 L-函数）控制，使得：
>
> 1. **谱 ⇔ 零点**：算术 CFT 的 scaling dimensions \(\Delta_n\) 与 Riemann zeta 零点 \(\rho_n = 1/2 + i\gamma_n\) 通过以下关系联系：
>    \[
>    \Delta_n - \frac{c}{12} = \frac{1}{4} + \gamma_n^2
>    \]
>    或等价地，\(L_0\) 在态 \(|n\rangle\) 上的本征值为 \(1/4 + \gamma_n^2 + c/12\)。
>
> 2. **配分函数 ⇔ L-函数**：算术 CFT 在 torus 上的配分函数与完备的 Riemann zeta 函数 \(\xi(s)\) 有直接的谱关系：
>    \[
>    Z_{\text{arith}}(\tau, \bar{\tau}) = \sum_n q^{\Delta_n - c/12} \bar{q}^{\bar{\Delta}_n - c/12} = \text{（某种对 }\xi(1/2 + i\gamma)\text{ 的逆 Mellin 变换）}
>    \]
>
> 3. **幺正性 ⇔ Riemann Hypothesis**：算术 CFT 是幺正的当且仅当所有 zeta 零点在临界线 \(\text{Re}(s) = 1/2\) 上。证明：幺正 CFT 要求所有 scaling dimensions \(\Delta_n \ge 0\)（实且非负）。由第 1 条，\(\Delta_n = 1/4 + c/12 + \gamma_n^2\)。如果 RH 成立，\(\gamma_n\) 为实数 ⇒ \(\Delta_n\) 为实数非负 ⇒ CFT 幺正。如果 RH 不成立，存在一个 \(\rho = 1/2 + \delta + i\gamma\)（\(\delta \neq 0\)），则对应的 \(\Delta\) 会有虚部 —— CFT 非幺正（有 ghost 态）。
>
> 4. **GUE 统计 ⇔ 大 c CFT 的普遍行为**：zeta 零点展现的 GUE 统计分布不是数论的孤例，而是中心荷 \(c \to \infty\) 的混沌 CFT 的高能谱必然展现的**普适类**行为。这解释了 Montgomery–Odlyzko 观察到的配对相关函数——它是 CFT 的高 scaling dimension 谱的普遍特征，就像 Wigner–Dyson 统计是量子混沌系统的普遍特征。

### 4.5 中心荷问题：Cardy 的幽灵

如果算术 CFT 存在，它的中心荷 c 是多少？

从物理直觉：系统的「大小」影响能谱密度。一个紧凑的系统有离散的、间距大的能谱。一个无限大的系统有连续的、无穷密的能谱。算术系统的「大小」是多少？素数有无穷多个——系统在「素数维度」上是无限的。每条周期轨道（每个素数 p）贡献一整个系列的态。这暗示 c → ∞。

但问题来了。CFT 的中心荷 c 通过 Cardy 公式控制高能态的渐近密度：
\[
\rho(\Delta) \sim \exp\left(2\pi \sqrt{\frac{c}{3} \Delta}\right) \quad (\Delta \to \infty)
\]

而 zeta 零点的密度由 Riemann–von Mangoldt 公式给出：
\[
N(T) = \#\{\gamma : 0 < \gamma \le T\} \sim \frac{T}{2\pi} \log \frac{T}{2\pi e}
\]

如果 scaling dimensions 是 \(\Delta_n \sim \gamma_n^2\)（如 §4.4 猜想），则态密度作为 Δ 的函数是：
\[
\rho(\Delta) \sim \frac{dN}{d\Delta} \sim \frac{\log \Delta}{4\pi \sqrt{\Delta}}
\]

这比 Cardy 公式的指数增长 **慢得多**——它是幂律衰减乘以对数，而非指数增长。

**这是整个猜想中最严重的张力。** 如果"算术 CFT"的态密度与任何已知 CFT 的态密度行为完全不同，称它为 CFT 到底意味着什么？有几种可能的出路：

1. **对数 CFT（Logarithmic CFT）**：在某些非幺正或对数的 CFT 中，Cardy 公式被修改。态密度可能出现多项式而非指数行为。如果算术 CFT 是对数 CFT（这与其非紧致的、无穷多态的性质可能相容），zeta 零点的 \(\sim T \log T\) 密度可能是自然的。

2. **Scaling dimensions 不是 \(\gamma^2\)**：直接的对应关系可能更复杂。零点的谱（imaginary parts γ）和 scaling dimensions 之间可能需要某种谱变换——可能是 Mellin 卷积或更复杂的积分变换。在这种情况下，\(\rho(\Delta) \sim \exp(\sqrt{c\Delta})\) 可能在 Δ 空间中是成立的，但在 γ 空间中表现为不同的形式。

3. **"CFT" 是误导性的**：算术系统可能是一个更一般的量子场论——不一定共形，不一定紧致，不一定满足 Cardy 公式的标准假设。可能性包括：
   - **Non-compact CFT**（如 Liouville 理论，其谱是连续的）
   - **TT̄ 变形的 CFT**（其态密度在有限变形参数下被修改）
   - **Sachdev–Ye–Kitaev (SYK) 类型的混沌量子力学系统**（在大 N 极限下态密度也是多项式的）

4. **温度参数不是 τ**：如果"算术温度"对应的模参数不是标准 torus 的 τ，而是某种其他周期结构的模参数，那么配分函数的 q-展开和 Cardy 渐近行为会不同。

**目前最诚实的态度是**：这个张力不意味着猜想是错的——它意味着"CFT"的标签可能是粗糙的。一个更精确的猜想应该指定算术系统属于哪一类量子场论（对数的？非紧致的？TT̄ 的？），并用该类场论的态密度公式来匹配 zeta 零点的 Riemann–von Mangoldt 密度。这正是 §7.1 中提出的 toy model 计算可能帮助诊断的问题。

## 5. \(\mathbb{F}_1\)：绝对零度

### 5.1 一元域——没有差异的地方

「一元域」\(\mathbb{F}_1\) 是这样一个对象上的域：它只有一个元素。所以 \(1 = 0\)。只有一个元素，意味着没有「非零」这个东西。没有差异。没有加法结构。没有乘法结构。

在物理上，这恰好是**绝对零度**：\(T = 0\)。

在绝对零度，热涨落完全消失。系统冻结在基态。没有激发。没有差异——所有微观态坍缩到一个纯粹的基态。

在 \(\mathbb{F}_1\) 上，没有「素数」——因为只有一个元素，没有什么可以区分的。所有有限域 \(\mathbb{F}_p\) 在某种意义上是 \(\mathbb{F}_1\) 的「热激发」——添加了 \(p\) 个元素后出现的结构。

### 5.2 从 \(\mathbb{F}_1\) 到 \(\mathbb{C}\) 的相变

从 \(\mathbb{F}_1\) 开始，我们「加热」——逐步增大 \(p\)。每增加一个素数 \(p\)，我们就激活了一整个系列的算术自由度：
- \(\mathbb{F}_p\) 上的点计数
- \(\mathbb{F}_p\) 上的 étale 上同调
- \(\mathbb{F}_p\) 上的 Galois 表示

在 \(p \to \infty\)（特征零）的极限，所有这些自由度同时被激活。系统抵达临界温度。此时：
- 组合数学（\(\mathbb{F}_1\) 上唯一的数学）被「融化」为连续的几何
- 有限域上的点计数（整数）变成复上同调的维数（仍然是整数——因为它们是极限）
- 有限域上的 Frobenius 迹变成某种复算符的迹（Hodge 理论中的 Frobenius？）

### 5.3 基态波函数

在绝对零度，系统处于基态 \(|\Omega\rangle\)。基态波函数 \(\Psi_0\) 编码了一切——所有激发都是对基态的微扰。

在 \(\mathbb{F}_1\) 上，「基态」是什么？由于没有任何结构，**基态必须是纯的组合/集合论对象**。可能是：
- **集合范畴**（拥有有限集合和函数）
- **有限集的同伦类型**（homotopy type of finite sets）
- 或更根本地：**存在的纯粹范畴**——对象是「存在」，态射是「存在之间的对应」，没有更多结构

有趣的是，\(\mathbb{F}_1\)-几何的当前候选理论（Borger 的 \(\lambda\)-环、Connes–Consani 的幺半群几何）**都**涉及某种「组合核心」加上额外的平滑结构。这就像在基态周围做微扰展开。

以 Borger 的 \(\lambda\)-环理论为例：一个 \(\lambda\)-环是一个交换环加上「Frobenius 提升」——对每个素数 \(p\)，有一个操作 \(\psi^p\)（Adams 操作）。这些 \(\psi^p\) 正是 **对基态的微扰**——每个素数 \(p\) 添加一个新的对称性操作，将 \(\mathbb{F}_1\) 的纯组合结构「加热」为丰富的算术结构。

在重整化群语言中：**Adams 操作 \(\psi^p\) 是素数 \(p\) 处的 relevant perturbation。** 沿着整个素数谱积分所有这些微扰，得到完整的 \(\mathbb{Z}\) 上几何——就像沿着 RG 流积分得到临界理论。

## 6. 狂热但可测试的预测

一个物理理论如果没有任何可测试的后果，就不能算理论——只能是故事。以下是这个框架的几条可能测试。

### 6.1 预测 1：算术 CFT 的 torus 配分函数

如果存在一个 \(c \to \infty\) 的 CFT 其谱包含 zeta 零点信息，那么**存在一个模形式，其 Fourier 系数从 zeta 零点构造**。

我们可以构造一个 toy model。取 zeta 零点的有限截断 \(\{\gamma_1, \ldots, \gamma_N\}\)，定义：
\[
\Delta_n = \frac{1}{4} + \gamma_n^2
\]
（暂时忽略 \(c/12\) 的位移）。构造：
\[
Z_N(\tau) = \sum_{n=1}^N q^{\Delta_n}
\]

问：当 \(N\) 很大时，\(Z_N(\tau)\) 是否展现出模不变性的迹象？即，\(Z_N(\tau)\) 和 \(Z_N(-1/\tau)\) 是否在某种意义下接近？

已知的事实：zeta 零点在模变换下的行为不是任意的。与通过 \(\tau \to -1/\tau\) 的模变换相关的某种对称性的存在性是 Riemann Hypothesis 的**等价形式**（通过函数方程，见 §4.3）。

**关键的障碍与诊断**：一个真正的 CFT 特征标 \(\chi(\tau) = \sum_n d_n q^{\Delta_n}\) 的系数 \(d_n\) 必须是**非负整数**（态数）。如果直接用 \(\Delta_n = \gamma_n^2 + 1/4\)，则 \(\gamma_n^2\) 是随机分布的超越数——\(Z_N(\tau)\) 作为 q-级数的展开系数不可能是整数。这有力地说明：**直接的对应 \(\Delta_n = \gamma_n^2\) 不可能完整——零点的分布（整体性质）与 scaling dimensions（局部标签）之间需要某种谱变换。**

因此这个 toy model 不是测试"猜想是否正确"，而是测试"猜想需要多大的修正"。一个更正确的构造应该找到某种方式，将 zeta 零点的信息编码进 \(Z_N(\tau)\) 的整数系数中——可能通过对零点做分箱（binning）、与已知模形式卷积、或某种 Poisson 求和型变换。

**可进行的计算**：数值地取前 1000 个 zeta 零点，构造 \(Z_{1000}(\tau)\)，检查它在模变换下的行为。如果它近似表现为权 0 的模形式（加上从有限截断来的误差），这是一个弱正面信号。更强的正面信号是：存在某种对零点的组合/deformation（如按素数因子分组），使得构造出的 q-展开系数逼近整数。

### 6.2 预测 2：Virasoro 代数的「算术实现」

在标准 CFT 中，Virasoro 生成元 \(L_n\) 满足：
\[
[L_m, L_n] = (m-n)L_{m+n} + \frac{c}{12} m(m^2-1) \delta_{m+n,0}
\]

如果 zeta 零点真的是某个 CFT 的谱，那么这些零点必须组织成 Virasoro 代数的最高权表示（或其在非有理情形下的推广）。

这意味着：**应该存在一个自然的「算术 Virasoro 作用」在对 zeta 零点索引的态空间上。** 具体来说，应该有一个操作 \(L_{-1}\)（平移）在零点之间移动，\(L_0\) 对零点给出本征值，等等。

**可能的候选**：zeta 零点的某种「导数」或「差分子」可能扮演 Virasoro 生成元的角色。特别地，\(L_{-1} \sim \frac{d}{ds}\) 在 zeta 函数上的作用可能与零点的某种递归关系有关。

如果能在 zeta 零点之间找到类似于 Virasoro 作用下 OPE 展开的行为——即使是近似的——这将是极其强烈的正面信号。

### 6.3 预测 3：共形 bootstrap 约束零点的分布

在 CFT 中，四点函数的 crossing 对称性对标度维数（scaling dimensions）的谱施加了极强约束。如果算术 CFT 是一个真正的 CFT，它的 four-point function 必须满足 crossing：

\[
\sum_{\mathcal{O}} C_{12\mathcal{O}} C_{34\mathcal{O}} G_{\mathcal{O}}(z, \bar{z}) = \sum_{\mathcal{O}} C_{14\mathcal{O}} C_{23\mathcal{O}} G_{\mathcal{O}}(1-z, 1-\bar{z})
\]

在 scaling dimensions \(\Delta_n = 1/4 + \gamma_n^2\) 的极限 \(n \to \infty\)：
- 高能态（大的 \(\gamma_n\)）在 crossing 方程中受 conformal block \(G_{\mathcal{O}}(z, \bar{z})\) 的渐近行为压制
- 这意味着**低 lying 零点的位置通过 crossing 方程约束高 lying 零点的统计分布**

**可进行的计算**：如果算术 CFT 类似于 \(c \to \infty\) 的广义自由场（generalized free field，GFF），其四点函数有简单的形式：
\[
\langle \mathcal{O}(z_1) \mathcal{O}(z_2) \mathcal{O}(z_3) \mathcal{O}(z_4) \rangle = \frac{1}{z_{12}^{2\Delta} z_{34}^{2\Delta}} + \frac{1}{z_{14}^{2\Delta} z_{23}^{2\Delta}} + \frac{1}{z_{13}^{2\Delta} z_{24}^{2\Delta}}
\]

这对应了算子乘积只有恒等算子交换的简单情况（mean field theory）。在 GFF 中，scaling dimensions 的谱完全由 \(\Delta\) 和多迹算子的组合决定。**如果算术 CFT 是一个 GFF（或接近 GFF），那么 zeta 零点的统计行为应纯粹由某个基础 scaling dimension \(\Delta_0\) 的多粒子态决定。**

验证这一点：取某个 \(\Delta_0\)，构造 GFF 的 scaling dimensions 谱，计算其 pair correlation，与 Montgomery 的 GUE 预测比较。

这也是为什么 GUE 出现的一个候选原因：已知 GFF 的多迹算子的 scaling dimensions 展现出 GUE 类型的能级排斥——这是量子场论中普遍的高能态统计行为，与具体模型无关。

### 6.4 预测 4：\(\mathbb{F}_1\) 上的态计数

回到第 3 节的猜测：模形式的 Fourier 系数 \(a_n\) 是某个 \(\mathbb{F}_p\) 模空间上的点计数在 \(p \to \infty\) 的极限。

考虑 Eisenstein 级数：
\[
G_{2k}(\tau) = \sum_{(m,n) \neq (0,0)} \frac{1}{(m\tau + n)^{2k}} = 2\zeta(2k) + \frac{2(2\pi i)^{2k}}{(2k-1)!} \sum_{n=1}^\infty \sigma_{2k-1}(n) q^n
\]

其中 \(\sigma_{k}(n) = \sum_{d|n} d^k\) 是除数和。

注意：Fourier 系数 \(\sigma_{2k-1}(n)\) 是**除数和**——对所有整除 \(n\) 的整数的 \(2k-1\) 次幂求和。

在「素数 = 温度」的图像中，除数和自然地出现：整除 \(n\) 的数恰好是那些**在 \(n\) 的素因式分解中出现的素数**。所以 \(\sigma_k(n)\) 在计数所有「构成」 \(n\) 的素数。不同的素数 = 不同的温度尺度 = 不同的能量贡献。除数和是对所有能量尺度的求和——就像比热是对所有被激活的自由度的求和。

这意味着：**Eisenstein 级数是「自由算术系统」的配分函数——在这个系统中，不同素数（不同温度）的自由度是独立的，没有「相互作用」。**

Cusp 形式——如 Ramanujan 的 \(\Delta\)——则是「相互作用算术系统」的配分函数。Ramanujan 的递归关系 \(\tau(p^{k+1}) = \tau(p)\tau(p^k) - p^{11}\tau(p^{k-1})\) 中的 \(p^{11}\) 是「相互作用强度」。

## 7. 在何处下铲子

以下是三个具体的、可以在当前数学水平上执行的计算性项目。

### 7.1 项目 A：构造 zeta-CFT 的 toy model

取前 \(N\) 个 Riemann zeta 零点（虚部为正）。定义「截断特征标」：
\[
\chi_N(\tau) = \sum_{n=1}^N q^{\gamma_n^2 + 1/4}, \quad q = e^{2\pi i \tau}
\]

问：
1. \(\chi_N(\tau)\) 和 \(\chi_N(-1/\tau)\) 在 \(N \to \infty\) 时的渐进行为如何？差值是否趋近于 0？
2. \(\chi_N(\tau)\) 的 Mellin 变换是否与 \(\xi(s)\) 有关系？
3. \(\chi_N(\tau)\) 在 \(\tau \to 0\)（高温极限）的行为是否符合 Cardy 公式的某种版本？

**可计算性**：这个计算只需要 zeta 零点的数值表（已知前 \(10^{13}\) 个）和基本的数值分析。不需要新数学。

### 7.2 项目 B：解码 Elliptic Curve CFT

取一个椭圆曲线 \(E/\mathbb{Q}\)。其 Hasse–Weil L-函数 \(L(E, s)\) 有一个函数方程：
\[
\Lambda(E, s) = N^{s/2} (2\pi)^{-s} \Gamma(s) L(E, s) = \pm \Lambda(E, 2-s)
\]

与 Riemann zeta 一样，\(L(E, s)\) 的非平凡零点在临界线 \(\text{Re}(s) = 1\) 上（BSD 猜想的一部分是模性 ⇒ 全纯延拓 + 函数方程，但 Birch–Swinnerton-Dyer 本身涉及中心值）。

对于 CM 椭圆曲线（如 \(y^2 = x^3 - x\)），其 L-函数与 Hecke 特征标相关，是模形式。这意味着**这个椭圆曲线的「算术 CFT」是一个有理 CFT**——有有限个 primary fields，特征标是经典的模形式。

对于非 CM 椭圆曲线，其 L-函数不与一个权 2 模形式简单相关（需要对称平方提升等）。这可能意味着其「算术 CFT」是**非有理**的。

**项目 B**：对于 CM 椭圆曲线，尝试将其 L-函数的零点明确地与某个已知有理 CFT 的特征标对应。这可能给出算术 CFT 的第一个精确例子。

### 7.3 项目 C：scaling dimensions 的 pair correlation

对每个已知的有理 CFT，计算其高 lying primary fields 的 scaling dimensions 的 pair correlation 统计。

预测：对于 \(c < 1\) 的 Virasoro 极小模型，scaling dimensions 的 pair correlation 应该表现出**泊松统计**（无排斥）——因为极小模型的谱是有限维的、高度结构化的、间距不规则的。

对于 \(c \to \infty\) 的极限（如 free boson），pair correlation 应该趋近于 GUE。

这个预测是可测试的。如果对于大 \(c\) 的 CFT，scaling dimensions 确实展示 GUE 统计，那么 zeta 零点的 GUE 行为就自动有了解释——**它们是大 c CFT 的高能谱的必然统计行为，与算术系统特有的性质无关。** 算术 CFT 的特殊性不在 GUE，而在具体的 scaling dimension **位置**（哪些特定数值出现）。

## 8. 致谢与警告

这份笔记中的猜想没有一个在当前数学中是严格定义好的，更不用说证明的。它们是一个孩子看到温度计和素数表后可能产生的疑问：「这两个东西为什么这么像？」

科学史上有过无数这样的疑问，大多数都是死胡同。但偶尔——非常偶尔——两个看起来完全不同的东西被证明是同一个东西的两个投影。热和机械运动。电和磁。空间和时间。

也许——仅仅是也许——算术和热力学也共享一个共同来源。

如果是这样，这个来源必须在某个地方留下指纹。这份笔记试图猜测这些指纹可能长什么样，以及在哪里可以找到它们。

**但警告**：这里几乎每一个方程都需要被大幅修正才能成为真正的数学。已知的最严重的缺口包括：

1. **整数系数问题（§6.1）**：真正的 CFT 特征标的 q-展开系数必须是整数（态计数）。如果直接用 \(\Delta_n = \gamma_n^2\) 构造配分函数，系数不可能是整数——零点的分布与 scaling dimensions 之间需要一种迄今未知的谱变换。

2. **Cardy 张力（§4.5）**：zeta 零点的态密度 \(\sim T \log T\) 与标准 CFT 的指数增长 \(\sim \exp(\sqrt{c\Delta})\) 完全矛盾。如果算术系统是一个 CFT，它必须是对数的、非紧致的、TT̄ 变形的，或属于某个尚未被分类的量子场论家族。

3. **模权 vs 中心荷（§3.4）**：模形式的权 k 和 CFT 的中心荷 c 是两个不同概念，这篇笔记早期的类比将二者混淆了。正确的对应应该是 c 而非 k 扮演"反常总量"的角色。

4. **Hecke 乘性的过度解释（§3.3）**：\(a_{mn} = a_m a_n\) 是系数之间的等式，而非单个系数的素数分解。物理图像需要更精细地利用 Hecke 算子理论。

5. **动力学的缺失**：模形式作为「比热」的类比虽然诱人，但目前没有动力学原理从「素数温度」中导出模变换律。\(\mathbb{F}_1\) 是否真的扮演绝对零度的角色，取决于 \(\mathbb{F}_1\)-几何是否有一天能发展出动力学，而非仅仅是定义。

**但这也是重点**：自然哲学的价值不在于给出正确答案，而在于提出正确的问题。如果你在读这份笔记时，某一刻觉得「这不可能对，但如果有千分之一的几率是对的，那么......」——那就值得继续想下去。
