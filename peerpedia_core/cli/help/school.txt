Show the most-followed researchers on the network.  This is the user
directory — use it to discover people worth following and papers worth
reading.  Think of it like a "most-cited authors" list.

HOW TO READ THIS HELP
  The examples below show real commands you can copy and paste.
  Everything you should type is shown — just replace the example values
  with your own.

    <angle brackets>   Replace with your own value.  Do NOT type the brackets.
                       peerpedia article show <id>
                       → you actually type:  peerpedia article show abc12345

    [square brackets]  This part is optional.  You can skip it entirely.
                       peerpedia article list [--mine]
                       → you can type:  peerpedia article list
                       → or:            peerpedia article list --mine

    --double-dashes    A "flag" (option) that changes what the command does.
                       --title means "the title comes next"
                       --scores means "the scores come next"
                       Some flags take a value (--title "My Paper")
                       Others work alone (--mine, --force, --feed).

  The prompt character > or $ in examples means "type this in your terminal."
  Don't type the prompt itself.


EXAMPLES
  # Top 20 users (default)
  peerpedia school
  # → School — Top Users by Followers
  #     1  Marie Curie    142
  #     2  Albert Einstein 98
  #     3  Niels Bohr      76
  #     ...

  # From a specific peer server
  peerpedia school --server https://peer.example.com
  # → School — Top Users by Followers
  #     1  Marie Curie    156
  #     ...

  # Show top 10
  peerpedia school --limit 10

  # Show top 50
  peerpedia school --limit 50

  # From local database only (no network)
  peerpedia school --local
  # Uses only your cached data. May be smaller or outdated
  # if you haven't synced recently.

  # School with JSON output
  peerpedia school --limit 5 --json
  # → [{"name": "Marie Curie", "follower_count": 142, ...}, ...]

HOW TO USE THE SCHOOL
  The users at the top have the most followers — they are often
  influential researchers whose papers are worth reading.  Follow
  a few to populate your feed with high-quality content.

  Follow someone from the school:
    peerpedia follow @top_researcher

  Then see their papers:
    peerpedia article list --feed

  Check the school on a specific peer server to see what the
  community on that server reads.

FLAGS
  --limit     How many users to show. Default: 20. Max: no limit
              (but output gets long). Shown in examples above.
  --server    Which peer server to fetch the school from. Default:
              PEERPEDIA_SERVER env var. Shown in example above.
  --local     Read from local database only — no network request.
              Default: off (fetches from server). Shown in example above.
  --json      Print output as machine-readable JSON. Default: off.
              Shown in example above.
