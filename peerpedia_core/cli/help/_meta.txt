PeerPedia has several ways to get help.  Pick what works for you:

  peerpedia --help               Full command list, grouped by what you want to do
  peerpedia <command> --help     Detailed options and examples for one command
  peerpedia mother               Step-by-step guide for new users
  peerpedia schema [command]     Machine-readable reference (for scripts and AI)

HOW TO READ THIS HELP
  The examples below show real commands you can copy and paste.
  Everything you should type is shown — just replace the example values
  with your own.

    <angle brackets>   Replace with your own value.  Do NOT type the brackets.
                       peerpedia article show <id>
                       → you actually type:  peerpedia article show abc12345

    [square brackets]  This part is optional.  You can skip it entirely.
                       peerpedia article list [--mine]
                       → you can type:  peerpedia article list
                       → or:            peerpedia article list --mine

    --double-dashes    A "flag" (option) that changes what the command does.
                       --help means "show help for this command"
                       Some flags take a value (--server https://...)
                       Others work alone (--json, --rich).

  The prompt character > or $ in examples means "type this in your terminal."
  Don't type the prompt itself.


EXAMPLES
  # What can PeerPedia do?
  peerpedia --help

  # I want to review a paper — how?
  peerpedia review submit --help

  # I'm new — walk me through the basics
  peerpedia mother

  # I want to find a specific command
  peerpedia --help | grep review

  # Show help in the REPL (interactive, with Rich formatting)
  peerpedia           # enter REPL, then type:
  :help create        # → detailed help in a styled panel
  :help review        # → all review subcommands grouped together

TIPS
  • Every error message tells you what went wrong and suggests what to try next.
  • Use @name to refer to people (like @einstein), title words for papers.
  • You don't need to memorize IDs — PeerPedia accepts partial matches.
  • Add --help to any command to see examples you can copy and paste.
  • In the REPL, :help <cmd> shows the same content in a styled panel.
