PeerPedia has several ways to get help.  Pick what works for you:

  peerpedia --help               Full command list, grouped by what you want to do
  peerpedia <command> --help     Detailed options and examples for one command
  peerpedia mother               Step-by-step guide for new users
  peerpedia schema [command]     Machine-readable reference (for scripts and AI)

EXAMPLES
  # What can PeerPedia do?
  peerpedia --help

  # I want to review a paper — how?
  peerpedia review submit --help

  # I'm new — walk me through the basics
  peerpedia mother

  # I want to find a specific command
  peerpedia --help | grep review

TIPS
  • Every error message tells you what went wrong and suggests what to try next.
  • Use @name to refer to people (like @einstein), title words for papers.
  • You don't need to memorize IDs — PeerPedia accepts partial matches.
  • Add --help to any command to see examples you can copy and paste.
